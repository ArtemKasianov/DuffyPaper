## Scripts for data analysis performed in the paper "Post-admixture selection favors Duffy negativity in the Lower Okavango Basin". 
> _PlotPCAs.r_ - PCA visualization

>  _VisualizeAdmixture.rfmix.py_ - Individual ancestry proportions visualization based on RfMix results

> _resampling.CLI.plots.py_ - Binomial resampling procedure implementation

> _Wright_Fisher_simulator.py_ - Implementation of simulations based on the Wright-Fisher model

> _Selection_coefficient_estimator.py_ - Selection coefficient estimator

> _CalculateSDStatsAndDrawPlotsForChrs.py_ and _CombineAncestryInfoByPopulations.py_ - Scripts used for preparing Figures 3 and S5

> _CalculateFadmValues.3pop.py_ and _DrawDistributionPlots.Fadm.py_ - Scritps used for preparing Figure S4

> _FilterCentromersTelomers.py_ - Scripts used for filtering telomeric and centromeric regions
