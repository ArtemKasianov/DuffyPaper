## Scripts for data analysis performed in the paper "Post-admixture selection favors Duffy negativity in the Lower Okavango Basin". 
> _PlotPCAs.r_ - PCA visualization

>  _VisualizeAdmixture.rfmix.py_ - Individual ancestry proportions visualization based on RfMix results

> _resampling.CLI.plots.py_ - Binomial resampling procedure implementation

> _Wright_Fisher_simulator.py_ - Implementation of simulations based on Wright-Fisher model

> _Selection_coefficient_estimator.py_ - Selection coefficient estimator
